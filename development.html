Development

Open Web App

The development of Vector Visualizer is separated into three parts: Proposal, Design, and Development & Testing.

Proposal

1. The Problem

Vector math problems involving basic linear algebra. Vectors, matrices, and the possibility for quaternions as well.

2. Purpose

It can be used to educate people of some concepts behind linear algebra. It can be used to investigate how many graphics programs use vector math to make things move on screen. Since it is also a full vector math library, it can be used for making games and other projects that involve vector math.

3. Audience

Adults and students alike. (And some really intelligent children too!)

4. Tools

HTML/CSS/Javascript.

5. Deliverables

Open source (Javascript) and internal documentation to go along with it.

Design

Environment

The environment will have a basic grid floor, have a mesh loaded in the center, and an easy-to-use interface for interacting with the mesh.

Gameplay

Although it is not a game, it still has first-person movement for flying around. This will allow users to see their manipulated mesh from any angle.

Art

The look and feel will be minimalistic. The design of the mesh itself will be, by default, simple as well. Of course, the mesh will be modifiable through the interface and .obj files will also be able to be loaded.

Sound and Music

There may be some ambient music playing in the background. It will have its own audio controller.

User Interface, Controls

The controls will be simple WASD, shift, and space movement. The user interface will be simple and easy to use HTML elements.
