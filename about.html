What is Vector Visualizer?

Vector Visualizer is an open source, free application built for the web. It is used as a learning resource and a hands-on experience with linear algebra. It is powered by WebGL and a custom-made vector math library which is also free and open source.

What features does Vector Visualizer include?

Vector Visualizer includes:

  • The editing of vertices and indices to form shapes and models.
  • The transformation of these shapes and models with translation, rotation, and scaling.
  • Basic .obj file support.
  • An open source vector math library for Javascript.

New features may be added and/or improved in the future.
